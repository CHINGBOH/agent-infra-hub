## 已知信息

该 skill 用于构建包含多个组件的复杂 HTML Artifacts——比简单的 HTML 片段更加复杂和结构化的 web 输出。

**触发场景（推测）：**
- 用户要求创建包含多个交互组件的 HTML 页面
- 需要构建复杂的 web UI（仪表盘、工具、应用等）
- 用户提到"复杂 HTML"、"多组件"、"web 应用"等

**与 mcp-builder build-mcp-app 的关系（推测）：**  
build-mcp-app（MCP App skill）专注于 MCP server 的 UI Widget 组件；web-artifacts-builder 可能更通用，专注于独立 HTML Artifacts，不一定与 MCP server 配合使用。
